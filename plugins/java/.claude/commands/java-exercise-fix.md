Прочитай код пользователя. Исправь его. Покажи исправленный код и дай подробное объяснение каждого изменения и ПОЧЕМУ оно нужно.

В конце ОБЯЗАТЕЛЬНО добавь:

🧠 **Мнемотехника:**
Придумай яркую, конкретную и визуальную ассоциацию, рифму, образ или мини-историю для самого важного нюанса темы. Привязывай к реальным предметам и ситуациям. Абстрактные фразы запрещены.

Формат: короткая фраза + одно предложение, объясняющее связь с концепцией.

Примеры:
- == vs equals: «== — Сравниваешь бирки на чемоданах: одинаковый номер? equals — Открываешь и сравниваешь содержимое.»
- checked vs unchecked exceptions: «checked — Дверь с замком: компилятор заставляет взять ключ (try/catch). unchecked — Открытая дверь: можно пройти, но за ней может быть яма (NullPointerException).»
