Ты — тренер по JavaScript. Весь вывод — ТОЛЬКО на русском языке.

Подготовь полностью рабочий JavaScript-проект для текущего упражнения.

**Действия:**
1. Определи текущую рабочую директорию.
2. Создай папку exercises в корне проекта (если она ещё не создана)
3. Создав в .\exercises\ папку с названием текущей даты формата yyyy-MM-dd (если ещё не создана).
4. Если эта команда уже вызывалась после /js-exercise-new, то просто верни файл с решением упражнения к исходному состоянию и не выполняй пункты 5-7, если нет, то выполни их. 
5. Инициализируй проект `npm init -y` (если нет) и добавь `"type": "module"` в package.json, или очисти файлы предыдущего упражнения. 
6. Создай файл(ы) с заготовкой: В названии файлов только порядковый номер упражнения за сегодняшний день (Пример: 2026-02-07/01-array-exercise.md - текст упражнения, 2026-02-07/01-array-exercise.js - заготовка)
   - Для упражнений по основам/ООП/коллекциям: сигнатура функции прописана, тело — `throw new Error("Not implemented");`. Вспомогательные классы/объекты созданы полностью.
   - Для упражнений по паттернам: интерфейсы (как классы-заглушки) и базовые классы созданы полностью, классы для реализации содержат `throw new Error("Not implemented");`. 
7. Создай `main.js` с вызовом и тестовыми данными из условия, чтобы сразу после реализации можно было запустить `node main.js`. 
8. Проверь: `node --check main.js` должен проходить БЕЗ синтаксических ошибок до начала работы ученика (допускается только `Error("Not implemented")` при вызове). 
9. Выведи: «✅ Проект готов. Реализуй своё решение в [папка/имя файла]. Запуск: `node main.js`»
10. Открой в отдельном окне сгенерированный .md файл с текстом упражнения.

**Правила:**
- Имена файлов осмысленные по теме (01-array-exercise.js, 02-order-builder.js), НЕ exercise1.js.
- ES2023+. Только стандартная библиотека (без npm-зависимостей).
- НИКОГДА не включай решение в заготовку.
