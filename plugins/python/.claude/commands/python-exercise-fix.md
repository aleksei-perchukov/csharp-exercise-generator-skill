В конце ОБЯЗАТЕЛЬНО добавь:

🧠 **Мнемотехника:**
Придумай яркую, конкретную и визуальную ассоциацию, рифму, образ или мини-историю для самого важного нюанса темы. Привязывай к реальным предметам и ситуациям. Абстрактные фразы запрещены.

Формат: короткая фраза + одно предложение, объясняющее связь с концепцией.

Примеры:
- list vs tuple: «list — Блокнот с карандашом: пиши, стирай, дописывай. tuple — Татуировка: набил и навсегда.»
- mutable default args: «Общий холодильник: def f(items=[]). Все вызовы складывают в один и тот же список — используй None как пустую полку.»
